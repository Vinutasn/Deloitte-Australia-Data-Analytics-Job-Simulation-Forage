# Deloitte Australia Data Analytics Job Simulation – Forage

This project was completed as part of the Deloitte Australia Data Analytics Virtual Experience Program offered through Forage.

The simulation focused on analyzing operational and workforce datasets using Power BI and Excel to generate business insights through dashboards and classification analysis.

---

# Task 1 — Machine Telemetry Analysis Dashboard

Built an operational dashboard using factory machine telemetry data to identify downtime patterns and monitor machine health across factories.

The source data was provided in **JSON format**, which was imported into Power BI and transformed by expanding nested data fields to prepare it for analysis and visualization.

Although the task mentioned Tableau, the dashboard was developed in **Power BI** based on existing hands-on experience with interactive dashboarding and reporting.

## Task Requirements
- Imported and transformed JSON data in Power BI by expanding nested records and preparing the dataset for analysis.
- Created a calculated measure field called **"Unhealthy"** with a value of 10 for every unhealthy status, representing 10 minutes of potential downtime since the previous message.
- Built a bar chart called **"Down Time per Factory"**.
- Created another bar chart called **"Down Time per Device Type"**.
- Designed an interactive dashboard where selecting a factory filters downtime results by device type.

## Tools Used
- Power BI
- Power Query
  
---

# Task 2 — Equality Score Classification Analysis

Analyzed employee compensation data using Excel to classify equality scores into business categories based on predefined conditions.

## Task Requirements
Created a new column called **"Equality Class"** using the following rules:

- **Fair** → Equality Score between -10 and +10
- **Unfair** → Equality Score less than -10 or greater than +10
- **Highly Discriminative** → Equality Score less than -20 or greater than +20

## Tools Used
- Microsoft Excel
